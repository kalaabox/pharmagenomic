Diabetes Data Analysis Report

About the Project

This project focuses on analyzing a dataset containing medical diagnostic data of patients to predict diabetes risk based on key health metrics like glucose level, blood pressure, BMI, and age. The objective was to explore the data, identify patterns, and build predictive models for healthcare research and patient risk assessment.

Dataset Details

The dataset, sourced from an open public repository, includes 768 records with 9 health-related parameters:

  • Pregnancies
  • Glucose
  • BloodPressure
  • SkinThickness
  • Insulin
  • BMI
  • Diabetes Pedigree Function
  • Age
  • Outcome (0 = Non-Diabetic, 1 = Diabetic)

Tools & Platforms Used

All analysis was performed on a Ubuntu Linux environment using:

  • Python (Pandas, NumPy, Matplotlib, Seaborn)
  • Jupyter Notebook
  • Command-line CSV processing

Steps Performed

  1. Downloaded dataset from online source
  2. Exploratory Data Analysis (EDA) and cleaning
  3. Calculated key descriptive statistics
  4. Visualized distributions and outliers
  5. Built logistic regression and decision tree models
  6. Evaluated model accuracy and visualized predictions

Result Highlights

The models achieved an accuracy of 78% in predicting diabetes status, with glucose and BMI emerging as the strongest predictors.
